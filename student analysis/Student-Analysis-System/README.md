# Student Marks Analysis

## Description
A beginner-friendly Python project that analyzes student marks using Pandas.

## Features
- Read student data from CSV
- Calculate grades
- Find average marks
- Save results to a new CSV file

## Technologies Used
- Python
- Pandas

## How to Run

1. Install Pandas

pip install pandas

2. Run the program

python analysis.py

## Output
Displays:
- Student details
- Grades
- Average marks

Creates:
- result.csv
